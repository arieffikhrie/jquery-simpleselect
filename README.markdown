jquery-simpleselect
===================

Custom style for HTML Select

<pre>
create a select on html
<code>
~~~~html
<select id="simpleselect">
  <option value="1">1</option>
  <option value="2">2</option>
</select>
~~~~
</code>
</pre>

<pre>
initialize when loaded
<code>
$("#simpleselect").simpleselect();
</code>
</pre>
